# Haskell Bazaar Backend

## Development

The development commands are defined in the `./backend/Makefile` file. Below is a list of the most useful commands for development:


* Start a ghci repl
```
make repl
```
* Lint the code
```
make lint
```
* Connect to the database with psql
```
make psql
```
* Run tests
```
make test
```
